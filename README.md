# Roommate Matcher

## Overview

Roommate Matcher is a C++ based Data Structures and Algorithms (DSA) project designed to help users find compatible roommates based on their preferences, lifestyle, and personal requirements.

The system collects user information and applies matching logic to identify the most suitable roommate candidates, demonstrating the practical application of DSA concepts and object-oriented programming.

---

## Features

- User Registration
- Preference-Based Matching
- Compatibility Analysis
- Roommate Recommendation
- Efficient Search Operations
- User Data Management
- Interactive Console Interface

---

- Algorithms

- Sorting Algorithms

---

## Project Workflow

1. User enters personal information.
2. User specifies roommate preferences.
3. System stores and processes data.
4. Matching algorithm evaluates compatibility.
5. Suitable roommate recommendations are displayed.
